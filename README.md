# Проект: Очереди и сортировка слиянием

Проект демонстрирует работу с очередью (Queue), моделирование обработки задач и сравнение производительности алгоритмов сортировки.


## Функциональность

Проект включает следующие компоненты:

### 1. Класс `Queue`

Реализует основные операции очереди:
* `enqueue(item)` — добавление элемента в конец очереди;
* `dequeue()` — удаление элемента из начала очереди;
* `is_empty()` — проверка, пуста ли очередь;
* `peek()` — просмотр первого элемента в очереди;
* `size()` — возвращает размер очереди.

### 2. Моделирование обработки задач

Функция `process_tasks(tasks)` моделирует процесс обработки задач, где каждая задача характеризуется:
* названием;
* временем выполнения (в минутах).

Программа выводит:
* время начала выполнения задачи;
* длительность выполнения;
* время завершения задачи.

### 3. Алгоритмы сортировки

Реализованы два алгоритма сортировки:

**Сортировка слиянием (`merge_sort`):**
* временная сложность: $O(n \log n)$ во всех случаях;
* пространственная сложность: $O(n)$;
* стабильная сортировка.

**Сортировка пузырьком (`bubble_sort`):**
* временная сложность: $O(n^{2})$ в среднем и худшем случае, $O(n)$ в лучшем случае (если список уже отсортирован);
* пространственная сложность: $O(1)$;
* стабильная сортировка.

### 4. Анализ производительности

Скрипт автоматически:
* генерирует случайные списки длиной 10, 100 и 1000 элементов;
* применяет к ним оба алгоритма сортировки;
* измеряет время выполнения;
* выводит сравнительную таблицу результатов.

## Требования

* Python 3.6 или выше.

## Анализ результатов

**Сортировка слиянием значительно эффективнее для больших наборов данных:**
* при 10 элементах разница минимальна или в пользу пузырьковой сортировки из‑за простоты алгоритма;
* при 100 элементах Merge Sort в ~18 раз быстрее;
* при 1000 элементах Merge Sort в ~128 раз быстрее.

**Рекомендации по использованию:**
* Merge Sort — оптимальный выбор для сортировки больших массивов данных;
* Bubble Sort — подходит только для учебных целей или очень маленьких списков (до 50 элементов).
